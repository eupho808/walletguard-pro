Questions

Everything you might ask.

Is it really free? What's the catch?

Yes, free. MIT licensed. The catch is that the AI cache uses OpenRouter if you enable it — that's optional, has its own free tier, and only runs on addresses you explicitly check. Everything else (phishing, simulation, drainer detection, MEV, EIP-7702, approvals) runs 100% locally with zero API keys.

How is this different from MetaMask's built-in alerts?

MetaMask uses Blockaid (closed-source, paid) and only shows alerts after the user confirms. WalletGuard intercepts before the wallet sees the transaction, decodes the calldata, runs simulation, checks 13 chains for existing approvals, scores against 29 layers — and shows you a full receipt. MetaMask shows a generic popup. We show what the transaction actually does.

Does it work with my wallet?

Yes — MetaMask, Rabby, Phantom, OKX, Brave Wallet, Frame, and any wallet that implements window.ethereum. We wrap the provider at the RPC level; your wallet never knows we're there.

Can it drain my wallet?

No. WalletGuard never holds keys, never signs transactions, and never has custody. All it does is read public RPC data and show you warnings. The revoke generator outputs calldata that you sign in your own wallet.

What about Ledger / Trezor / hardware wallets?

Hardware wallets are supported with extra strict rules. We verify the recipient matches what your screen shows, flag any address substitution attempt, and require explicit confirmation for unlimited approvals.

What data leaves my browser?

Only what you explicitly send. By default: nothing. If you enable the AI cache (optional), OpenRouter sees addresses you check. If you enable the community threat feed (optional), GitHub serves you a signed manifest — no telemetry sent back.

Has it been audited?

Self-audited (full report in SELF_AUDIT.md). 19 bugs found and fixed during v3.2.1 → v3.3.0 cycle. External audit is the next milestone — see THREAT_MODEL.md for residual risks and what's in scope.

Why not just use Pocket Universe?

Pocket Universe was excellent and was acquired by MetaMask in 2023 for an estimated $5-15M. After acquisition, the standalone extension was deprecated. WalletGuard Pro is the open-source successor — same simulation engine, but 29 layers, 13 chains, EIP-7702 detection, blast radius, drainer DNA, correlation, price oracle, ENS resolution, and threat feed.

What's new in v3.6 "PORTFOLIO"?

WalletGuard now answers the three questions every wallet user asks: "What do I own?" — portfolio view with USD totals across all chains (real-time Uniswap V3 prices). "What's exposed?" — existing approval scanner + blast radius. "How do I fix it cheaply?" — bulk multicall revoke UI that turns N stale approvals into K transactions (K = unique chain/token pairs) using Multicall3 aggregate(). 1,225 tests across 30 suites pass. No other open-source extension ships all three in one MIT-licensed package.

What's new in v3.5 "FORTRESS"?

Six new modules: USD blast radius (real-time prices via Uniswap V3), native ENS resolution (pure-JS Keccak-256, no APIs), stale approval tracker (180d threshold, waste-score ranking), wallet classifier (8 types with adaptive rules), exportable audit log (CSV/JSON, privacy-first), and bulk multicall revoke (N approvals → 1 tx). Plus 3 world-first detection systems from v3.4: drainer DNA matching, correlation engine, and real-time blast radius.
