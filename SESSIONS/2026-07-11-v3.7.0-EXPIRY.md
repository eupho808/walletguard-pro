## Project conventions

- `npm test` runs all suites in sequence
- `node build.js` rebuilds content.js + popup-bundle.js
- `node build-firefox.js` + `node build-firefox-pack.js` → Firefox ZIP
- `node build-chrome-pack.js` → Chrome ZIP
- All bundle artifacts must pass `node --check`
- Build ORDER in `build.js` lists modules in dependency order
- Test-build.js verifies all expected modules exposed in `WG_POPUP_LIB`
- Storage keys centralized in `STORAGE_KEYS` object in background.js
- SENSITIVE_KEYS set lists keys omitted from settings export
- Locale files use unicode escapes (except ja.js which uses actual Japanese chars)
